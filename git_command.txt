1. git init
   - ایجاد یک مخزن جدید Git در دایرکتوری فعلی.

2. git clone [repository-url]
   - کپی کردن یک مخزن موجود به صورت محلی.

3. git status
   - نمایش وضعیت فعلی مخزن و فایل‌های تغییر یافته.

4. git add [file]
   - اضافه کردن فایل‌های تغییر یافته به ناحیه staging.

5. git commit -m "[message]"
   - ذخیره تغییرات با یک پیام توصیفی.

6. git pull origin [branch]
   - دریافت آخرین تغییرات از مخزن راه دور و ادغام آن‌ها با شاخه محلی.

7. git push origin [branch]
   - ارسال تغییرات محلی به مخزن راه دور.

8. git branch
   - نمایش لیست شاخه‌های موجود.

9. git branch [branch-name]
   - ایجاد یک شاخه جدید.

10. git checkout [branch-name]
    - سوئیچ کردن به یک شاخه دیگر.

11. git merge [branch-name]
    - ادغام تغییرات یک شاخه دیگر به شاخه فعلی.

12. git log
    - نمایش تاریخچه تغییرات و کامیت‌ها.

13. git diff
    - نمایش تغییرات بین فایل‌های تغییر یافته و آخرین کامیت.

14. git reset [file]
    - حذف فایل از ناحیه staging.

15. git rm [file]
    - حذف فایل از مخزن و ناحیه staging.
